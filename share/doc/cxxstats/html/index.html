cxxstats

Contains both a library and an executable.

The executable is called cxxstats, which you can use for working with your own stats

Linking

Assume you have installed it autogen.sh creates the default install dir of ~/.local), your program should have the include line: 
#include <mcstats.h>
Link against the library 
libcxxstats
thus: -lcxxstats

Functions in the library

You can call the functions from your own program, assuming that you have linked as instructed above.

sortd

Sort doubles in place

quantile

Caclualtes quantiles according to Excel interpolation formula. Assumed array is sorted.

basic_stats

Returns structure of basic stats: length, mean, sample stdev

frank

Fractional rank, as detailed on Wikipedia.
